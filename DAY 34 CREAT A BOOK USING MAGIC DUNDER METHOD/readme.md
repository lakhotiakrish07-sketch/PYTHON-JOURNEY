# Day 34 - Magic (Dunder) Methods in Python

Today I practiced Magic Methods in Python by creating a Book class.

## What I Implemented

- __init__ → Initializes object attributes
- __str__ → Custom print output
- __len__ → Returns number of pages
- __add__ → Adds pages of two books

## Project Description

I created a Book class where:
- Printing the object shows book details
- Using len() returns total pages
- Adding two book objects adds their pages

## Key Learning

Magic methods allow us to define how objects behave with built-in functions like:
- print()
- len()
