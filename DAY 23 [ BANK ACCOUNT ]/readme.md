# Bank Account System using Constructor (Python)

This project is a simple demonstration of Python Object-Oriented Programming.
It focuses on the use of constructors (__init__) to initialize object data.

## Concepts Used
- Classes and Objects
- Constructor (__init__)
- Instance variables
- Basic methods

## Project Overview
The program creates:
- An Account class to store account details
- An Operation class to calculate available balance

Account information and balance are handled using objects.

## Purpose
This project was created to practice Python constructors and understand
how data is initialized and accessed within classes.
