# Day 22 – Student Details Program (OOP Basics)

## Description
This is a simple Python program created to practice the basics of
Object-Oriented Programming using **classes and objects**.
The project does not use constructors and focuses on understanding
how object variables and methods work.

## Concepts Used
- Class
- Object
- Methods
- Object variables
- User input and output

## How It Works
- A `Student` class is created
- An object of the class is instantiated
- Student details are taken using user input
- Details are displayed using a class method

## How to Run
1. Make sure Python is installed
2. Run the program using:
   python main.py

## Learning Outcome
- Understand the relationship between class and object
- Learn how to access and modify object variables
- Practice calling methods using objects
