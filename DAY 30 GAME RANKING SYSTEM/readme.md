#  Game Ranking System (Day 30)

This is a simple Python project created to understand **Class Methods** in Object-Oriented Programming.

##  Concept Used

- Class
- Constructor (`__init__`)
- Instance Variables
- Class Variable
- Class Method (`@classmethod`)

---

##  Project Idea

This project simulates a small game ranking system.

- Each player has:
  - Name
  - Level
  - Score
- The game has a global difficulty rank.
- The difficulty can be changed using a class method.

---

##  How It Works

- `game_rank` is a class variable.
- `level_change()` is a class method.
- When the difficulty is changed, it updates for the entire class.

---

## ▶ Example Flow

1. Create a player.
2. Display details.
3. Change game difficulty.
4. Create another player.
5. Check updated difficulty.
