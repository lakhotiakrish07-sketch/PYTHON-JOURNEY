# Utility Toolkit - Day 28

This project demonstrates the use of Object-Oriented Programming (OOP) in Python using static methods.

## Concept

The Utility class contains common mathematical and temperature conversion functions.  
Static methods are used because these functions do not depend on instance variables.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Fahrenheit to Celsius conversion
- Celsius to Fahrenheit conversion

## Why Static Methods?

Static methods are used when a function:
- Does not use instance variables
- Does not use class variables
- Belongs logically inside a class

This helps in organizing related functions in a structured way.
