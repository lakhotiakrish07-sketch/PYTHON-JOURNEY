# Day 33 - Python super() Keyword

## Topic
Understanding and using the `super()` keyword in Python OOP.

## Project Description
In this project, I created a simple Vehicle system using inheritance.

- `Vehicle` is the parent class.
- `Car` is the child class.
- I used `super()` to call the parent class constructor inside the child class.

## What I Learned

- How inheritance works in Python
- How to use `super()` to access parent class constructor
- How to avoid repeating code
- How parent and child classes connect

## Output
Vehicle constructor called  
Car constructor called  
Brand: Toyota  
Speed: 180 km/h  
Model: Fortuner  

## Conclusion
The `super()` keyword makes inheritance clean and professional.
It helps in calling parent class methods easily.
